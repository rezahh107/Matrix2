# راهنمای توسعه دهندگان

## توسعه سیستم مدیریت مراکز

### معماری

```
Core Layer (مرکز منطق کسب‌وکار)
├── center_manager.py (ادغام تنظیمات)
├── allocate_students.py (الگوریتم تخصیص)
└── policy.py (مدل‌های داده)

UI Layer (رابط کاربری)
├── main_window.py (بخش مدیریت مراکز)
└── app_preferences.py (مدیریت تنظیمات)

CLI Layer (خط فرمان)
└── allocate.py (پارسر آرگومان‌ها)
```

### افزودن مرکز جدید

1. **فقط Policy را تغییر دهید** - بدون نیاز به تغییر کد
2. **سیستم به طور خودکار تشخیص می‌دهد**
