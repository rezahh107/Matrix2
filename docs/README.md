# مستندات Core Policy Adapter

- نرمال‌سازی ورودی در لایهٔ Reader ارقام فارسی/عربی و هدرها را یکسان می‌کند.
- تست پذیرش `tests/test_allocation_system.py` تضمین می‌کند تغییر JSON بدون تغییر کد رفتار را عوض می‌کند.
